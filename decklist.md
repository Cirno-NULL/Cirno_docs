# 外置牌堆列表

![](.gitbook/assets/sheng-dan-mao-qi-lou-nuo-.png)

{% hint style="info" %}
牌堆抽取方法：.draw+空格+你想要抽取的牌堆

示例如下：
{% endhint %}

```text
.draw 大成功
.draw 地名
.draw 炉石经典卡包
.draw 塔罗牌单卡
.draw 验证码
.draw 密教
```
